# Free Resume Builder

A professional, fully client-side, serverless resume builder. Build your resume, see live changes, and export to PDF—without ever leaving your browser.

## 🚀 Quick Start

1. `npm install`
2. `ng serve`
3. Open `http://localhost:4200`

## 🛠 Tech Stack

- **Framework**: Angular v21 (Standalone)
- **Styling**: Tailwind CSS
- **Rich Text**: ngx-quill
- **PDF Engine**: html2pdf.js
- **Testing**: Vitest
