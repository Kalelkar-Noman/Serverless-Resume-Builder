## [v1.0.0]

### Added

- **Serverless Resume Builder**: A fully client-side resume builder for creating ATS-friendly resumes online.
- **Client-Side PDF Generation**: Generate pixel-perfect PDFs directly in the browser without server roundtrips.
- **Auto-Typing Preview**: A dynamic landing page feature that showcases the builder in action.
- **Resume Parser**: Import and parse existing resumes to seamlessly switch formats.
- **Live Preview**: Real-time side-by-side scaling and updating of the resume preview while typing.
- **Documentation Suite**: Comprehensive Diátaxis and arc42 documentation for contributors.
