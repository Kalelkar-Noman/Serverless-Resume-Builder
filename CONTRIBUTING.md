# Contributing to Free Resume Builder

Thank you for your interest in contributing! We welcome improvements, bug fixes, and new resume templates.

## 🚀 Getting Started

1. **Fork** the repository.
2. **Clone** your fork locally.
3. **Install** dependencies: `npm install`
4. **Create a branch**: `git checkout -b feat/your-feature-name`

## 🛠 Development Workflow

This project uses an automated pipeline to ensure code quality. Before submitting a pull request, ensure your changes pass the following checks:

### 1. Formatting & Linting

We use **Prettier** and **ESLint** to maintain consistent code quality. Run these to check your files:

```bash
npx prettier --check "src/**/*.ts"
npx ng lint
```
